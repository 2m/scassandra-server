Setting up the project
-----------------------

This project uses SBT, which can be downladed and installed from here: http://www.scala-sbt.org/release/docs/Getting-Started/Setup.html

Once installed, go to the root folder of this project and run:

> sbt

This should download and setup all the project dependencies and take you to the SBT interactive console.

You then can then run

> gen-idea

To generate IntelliJ IDEA project files

and/or

> exit

To exit the interactive console.
